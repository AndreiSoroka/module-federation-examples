# vue-cli@5.x.x Module Federation

Init and start first app
```bash
cd app-exposes;
npm i;
npm run serve;
```

Init and start second app
```bash
cd app-general;
npm i;
npm run serve;
```
After that open
http://localhost:8081

P.s. app-exposes have a temporary fix.
